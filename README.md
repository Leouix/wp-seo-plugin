# wp-seo-plugin

Плагин для сео-настроек сайта Wordpress. 
- Запрещает пагинацию на страницах и редиректит с https://domain.com/page/n на https://domain.com/
- Редиректит на нижний регистр урла: https://domain.com/?get=SomE на https://domain.com/?get=some
- Удаляет из урла повторяющиеся дефисы с https://domain.com/some----post на https://domain.com/some-post
